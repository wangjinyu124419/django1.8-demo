{% trans 'Why I see the latency Dialog ?' %}

PPS&PushCMS rely on several third party api services, when these service down, you may get problem with some function.

When these third party service's latency too high, PPS will show a dialog to notify you to pay an attention.

All services PPS&PushCMS relay on as following:

LCX-PushServer

{% trans 'Current site.' %}

DW-ImageServer（http://img.transcoder.opera.com/）

{% trans 'The image download server.' %}

{% trans 'If down, you may not see some or all of the pictures.' %}

Solution: Waiting for server back to normal or ask Transcoding-Team for support.

UP-ImageServer（http://img-src.transcoder.opera.com/）

{% trans 'The image upload server.' %}.

{% trans 'If down, you may not upload picture or edit news image.' %}

Solution: Waiting for server back to normal or ask Transcoding-Team for support.

IN-TransServer

{% trans 'India news profile transcode server.' %}

{% trans 'If down, you may get nothing when click the "transcode" button to add India news.' %}

Solution: Waiting for server back to normal or ask Infra-Team for support.

{% trans 'Indonesia news profile transcode server.' %}
